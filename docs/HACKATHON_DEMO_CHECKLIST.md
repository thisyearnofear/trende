## Best Demo Path

1. Start on the Trende landing page.
2. Explain the problem in one sentence.
3. Create or show a market tied to a real topic.
4. Show the Chainlink CRE workflow source briefly.
5. Show the app processing state.
6. Show the final conviction result.
7. Show the on-chain oracle outcome.
8. Close with the agent-economy use case.
